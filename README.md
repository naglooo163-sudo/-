Вот структурированный сценарий вводного урока по Java. Он рассчитан на полного новичка (возможно, знакомого с другими языками) и разбит на логические блоки с таймингом.

Тема: «Погружение в Java: от структуры класса до ветвлений»

Длительность: 60 минут

Аудитория: Новые сотрудники (Junior-разработчики, стажёры)

Цель: Объяснить, как запускается Java-программа, что такое класс и метод `main`, показать синтаксис переменных, типов данных и условных операторов.
